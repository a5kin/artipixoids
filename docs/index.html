    The Past

    Artipixoids! is half generative art, half artificial life project with a focus on genetic cellular automata obeying energy conservation principle. Started in the end of 2014, it has undergone several major modifications, and is still in development. Below, you can see an antology of different modifications.

    The Future

    The project would be split into 3 main parts.

    HECATE

    HECATE stands for Hermetically-Evolutionary Cellular Automata Template Engine. This would be the core framework, allowing us to build new CA modifications and run them on different parallel hardware.

    {% endcomment %}

    MOIRE

    HECATE would be covered in MOIRE, which is Multi-Objective Interactive Runtime Environment. Generally it is intended to be the user interface, allowing to run HECATE models interactively.

    VAULT

    There could be also a public place to keep and share all HECATE mods over the world. VAULT is a good name for it. But honestly, we are not yet sure how it may be organized.
